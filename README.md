# Crowdfunding App

## Prerequisites
1. **XAMPP Installed**: Ensure you have XAMPP installed.
2. **Start MySQL**: Open XAMPP Control Panel and click **Start** next to MySQL.

## Setup
The application is configured to connect to MySQL on `localhost:3306` with user `root` and no password (standard XAMPP default).
It will automatically create the database `crowdfunding_db` if it doesn't exist.

## Running
Run the following command in the terminal:
```powershell
.\mvnw.cmd spring-boot:run
```

Access the app at: [http://localhost:8080](http://localhost:8080)
